# Getting started

Install the .NET Core SDK and lots of documentation: [https://www.microsoft.com/net/download/core](https://www.microsoft.com/net/download/core)

Build on OS X and Linux:

- `./build.sh build.cake`

Build on Windows:

- `./build.ps1 build.cake`

Run the source
- `cd src/Vending`
- `dotnet run`
